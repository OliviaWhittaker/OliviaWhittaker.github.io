Dolphin Print Design

I created this dolphin print in different colourways.
As pink dolphins exist in real life, I decided to
create a print using pink dolphins as the theme.

Firstly I sketched the design and traced it onto
thin paper.I then scanned in the image and
filled in the colour and created the print formation
using Adobe Photoshop. I think that this print would
good as a stationary design, fashion and accessory
print design wrapping paper design etc.
